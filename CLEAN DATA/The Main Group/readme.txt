1)“NoPersonalInfo.xlsx” Data with no personal information 
2)“TotalPaymentUsers.xlsx”, Each Row represents a different user ID, the multiple rows for each user was merged into one row per user .    The column   “PaymentAmount ” is the total number of payments for each user . The column “NuberPayments” (should be NumberPayments) is the number of payments each user made. The rest of the columns are the information for each user ID  that appears to be relatively constant of a user ID. If values where different for a particular user ID the most frequent value was used to fill in the ID.  The rest of the rows are left out.
